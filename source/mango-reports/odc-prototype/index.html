Introduction

Mango reports is a web-based reporting software under development.

It would include the following features:

  • Data integration from various sources (databases, web pages, spreadsheets...)
  • Data cleansing - data quality features
  • Computation of standard statistic measures and metrics using user-defined formulas
  • Generation of interactive reports and visualizations
  • Aggregate awareness (navigation inside tables automatically grouping hierarchic data)
  • User friendly user interface
  • Instant loading of pages, in web format (no Flash)
  • Robust security layer
  • Accessible via mobile devices
  • ...

Right now the project is at it's very beginning. Focus has been put on a generator of dynamic pivot tables, visible in some of the demos. A basic report generator should be delivered in the coming weeks.

This website is just a few hours work to present progress. It is not representative of the interface that will give access to the reports.

Prototype dashboards and interactive reports

Several outputs have just been drafted and submitted to the open data challenge.

Interactive reports demonstrate possibilities of tridimensional pivot tables with aggregate navigation. Such visualization is ideal for data analyst who need the figures but also need to detect trends, peaks easily.

  • Turnover index for some services activities
  • Waste generation report
  • Waste treatment report

Dashboards summarize large volume of data into meaningful figures. Such document is rather visual to present data to the public, or high-level people in companies.

  • Public data dashboard
  • Waste dashboard

You must have javascript enabled in order to have a convenient experience.

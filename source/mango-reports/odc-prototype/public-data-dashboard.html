Updates

This document shows figures based on an extraction of publicdata.eu catalog from the 15th of May 2011.

Figures might be updated in a near future, using another data source (ckan.net)

Data sources

The dataset has been extracted from the crawl cache powering publicdata.eu

Data processing and data quality

A filter has been applied to get only datasets belonging to European countries, since some datasets from other continents were also present. Unfortunately some datasets were missing the 'spatial' information and have been excluded even though they were belonging to European countries. More than 26 000 datasets could be identified.


This software is a prototype and figures are presented for demonstration purpose only. We cannot guarantee that the figures displayed are absolutely correct. If you need to perform data analysis for professional purpose then we recommend to use other tools until we can provide you with a stable version. follow-us on twitter or contact us to stay updated.

Introduction

This dashboard presents statistics about datasets referenced by the website http://publicdata.eu/.

You can visualize here figures about countries releasing the most data to the public, the licenses usually used, and the format on which the data is available.

Some countries referenced by http://publicdata.eu/ website Could not be taken into account in these statistics because of missing standard metadata specifying location.

For the countries presented here, some datasets were missing information about license or files formats.
